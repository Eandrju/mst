# mst
Bunch of MST algorithms and benchmark scripts


## python scripts 
 - **benchmark.py** - Automates the proccess of comparing algorithms. It generates sample graphs and run each algorithm on it.
 - **visualizer.py**  - Visualizes the results creating scalling diagrams
 - **validate.py** - Helpful script at the start of project. It uses python graph library networkx to validate whether result
 that I obtained was in fact correct.
 
## cpp sources
- **mst.cpp** - main file
- **createGraph.cpp** - name is pretty self-explanatory
